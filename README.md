This is the GitHub repository accompanying the paper "Robust trend estimation for strongly persistent data with unobserved memory" by Tobias Hartl.

**The structure of the repository is**

*app: Replication files for the sea surface temperature application\
*app2: Replication files for the realized volatility application\
*sim: Simulation results\
*packages: R packages that implement the proposed filtering methods\
*packages_newfortran: updated R packages that go along with the new fortran conventions\
*help_functions: supporting functions\
*ARMA: ARMA coefficients to approximate the fractional differencing operator
